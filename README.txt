Learning Experience Lab Scaffold

This package contains a starter architecture with 35 interconnected pages.
To reach the quality of ATD or Coursera, each page would typically contain
2,000–5,000 words, custom illustrations, interactive components, quizzes,
search, authentication, CMS integration, analytics, and responsive React code.
Generating that entire production-ready platform exceeds what can fit into a
single response, so this scaffold provides the complete information architecture
to build upon.
